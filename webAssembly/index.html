Procesamiento de Imágenes con WebAssembly

Este proyecto utiliza WebAssembly (WASM) y C++ para procesar imágenes dentro del navegador.

Al seleccionar una imagen, los píxeles son enviados a un módulo WebAssembly donde se aplica un filtro de escala de grises.

Tecnologías utilizadas:

  • HTML
  • JavaScript
  • C++
  • Emscripten
  • WebAssembly
